# SocketIO chat test

## Purpose

Create a basic chat server/client website with nodejs, jquery, express and socket.io server, to get a better understanding of how socket.io works.

## Features

* Server/Client  
* Chat interface with, active, idle and sleep icons
* Select Users
* Private Message
